# FeedFinder
Find Common RSS/Atom Feeds From A URL, Domain, or a File that Lists These

![Help Menu](https://cdn.discordapp.com/attachments/1434270963977031690/1437674310947831808/2025-11-10_23-22-15.png?ex=69141a0a&is=6912c88a&hm=dc101107dda87021ce844fe7ba89768a569da54fef5ad903228993a221126ace&)

### Information
Searches all Hosts in Parallel without Spamming any Single Host    
Searches are made by sending a HEAD request and parsing the response code and the content-type header   

### Finds:
- Wordpress - /feed/
- Shopify - /collections/all.atom
- Tumblr - /rss
- Blogger - /feeds/posts/default
- Random - /feed.xml & /rss.xml

(note that these patterns arent solely specific to their listed platform)
